# HappyIM Web 前端设计

## 整体布局

Web 端采用**三栏布局**，仿微信桌面版 / Slack 风格：

```
┌──────┬────────────┬──────────────────────────┐
│ 导航 │  列表面板   │      内容面板              │
│ 60px │  250px     │      剩余宽度              │
├──────┼────────────┼──────────────────────────┤
│      │            │                          │
│  💬  │ 会话1      │  聊天消息区域              │
│  👥  │ 会话2      │                          │
│  ⚙   │ 会话3      │  ┌──────────────────┐   │
│      │            │  │ 输入框            │   │
│      │            │  └──────────────────┘   │
└──────┴────────────┴──────────────────────────┘
```

## 页面路由

独立 URL 路由，保证浏览器前进/后退、刷新不丢状态、支持多标签页：

```
/login                          # 登录页（全屏居中表单）
/register                       # 注册页（全屏居中表单）
/chat                           # 主界面 - 消息Tab，右栏空白占位
/chat/:conversationId           # 主界面 - 消息Tab，打开指定会话
/contacts                       # 主界面 - 通讯录Tab，右栏空白占位
/contacts/:userId               # 主界面 - 通讯录Tab，查看联系人详情
/settings                       # 主界面 - 设置Tab
```

### 路由对应三栏状态

| URL | 左栏(60px) | 中间栏(250px) | 右栏 |
|-----|-----------|--------------|------|
| `/chat` | 消息图标高亮 | 会话列表 | 空白占位 |
| `/chat/p_userA_userB` | 消息图标高亮 | 会话列表（选中userB） | 聊天窗口 |
| `/contacts` | 通讯录图标高亮 | 通讯录面板 | 空白占位 |
| `/contacts/user001` | 通讯录图标高亮 | 通讯录面板（选中user001） | 联系人详情 |
| `/settings` | 设置图标高亮 | 设置菜单 | 对应设置内容 |

## 消息去重

后端用 80-bit 消息 ID 天然保证唯一（详见 [MESSAGE_ID_DESIGN.md](MESSAGE_ID_DESIGN.md)），MongoDB `message_id` 唯一索引兜底。前端只处理一种场景：**WebSocket 实时消息和上线游标拉取可能重叠**，用 message_id Set 去重即可：

```javascript
const renderedMessageIds = new Set();

function onReceiveMessage(msg) {
    if (renderedMessageIds.has(msg.message_id)) return;  // 已渲染，跳过
    renderedMessageIds.add(msg.message_id);
    messages.push(msg);
    messages.sort((a, b) => a.created_at - b.created_at);
}
```

## 技术要点

| 模块 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue 3 + Vue Router | SPA，独立路由 |
| 构建 | Vite | |
| WebSocket | 原生 WebSocket | 连接 + 心跳 + 重连 |
| HTTP | fetch / axios | API 调用 |
| 文件上传 | MinIO | 图片/文件对象存储 |
| 状态管理 | 组件内状态 + provide/inject | 轻量，先不用 Vuex/Pinia |
